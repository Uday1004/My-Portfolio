## Installation and Usage 🚀

To use this app locally, follow these steps:

Clone the project

```bash
  [https://github.com/Uday1004/My-Portfolio.git]
```

Go to the project directory

```bash
  cd My-Portfolio.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
